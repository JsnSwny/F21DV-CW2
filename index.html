GitHub Users Visualiser

This interactive visualisation shows the distribution of GitHub users around the world. The dataset consists of 170k users 10k unique locations.

  • Select a point to see the following relationships from other points.
  • Select a language to only show users that use that language.
  • Select a country to filter users by that specific country.
  • Drag date range graph to filter users by specific time range.
